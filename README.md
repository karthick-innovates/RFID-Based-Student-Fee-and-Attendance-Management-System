# 📚 RFID-Based Student Fee and Attendance Management System

An IoT-enabled system that automates **student attendance tracking** and **fee management** using **RFID technology**, **NodeMCU ESP8266**, and **MongoDB**. The project reduces manual work, improves accuracy, and provides real-time access to student records for educational institutions.

---

## 📖 Overview

The **RFID-Based Student Fee and Attendance Management System** is designed to simplify administrative tasks in schools and colleges by integrating RFID technology with IoT and cloud-based databases.

Each student is assigned a unique RFID card. When the card is scanned at an RFID reader:

- Attendance is recorded automatically.
- Student fee information is retrieved instantly.
- Pending fee notifications can be displayed before reaching the fee counter.
- Cashiers can update fee payment status in real time.
- Attendance records are stored securely in the database.

The system eliminates manual attendance registers, reduces queue congestion at fee counters, and provides an efficient, accurate, and scalable management solution.

## ⚙️ System Workflow

1. Student scans RFID card.
2. RFID reader reads the card's unique ID.
3. NodeMCU ESP8266 processes the request.
4. Student information is fetched from MongoDB.
5. Attendance is marked automatically.
6. Fee payment status is displayed.
7. Cashier updates payment records when fees are paid.
8. Database is updated in real time.
